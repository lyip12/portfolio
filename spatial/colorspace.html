COLOR SPACE . CS

A “shadow art”sculpture, a three-dimensional object or collection of objects, that when lit / projected from particular angles create different variations of projections.Inspired by the idea of construction and deconstruction of images and projections, this project utilizes the fundamental system for representing the colors to be used on a computer display, RGB, we created a program where the user can input their image, and the result is a spherical polygon-cloud which projects individual RGB images from specific views.

This was a collaboration between Lucy Yip and Yiqi Zhao.
My contribution to this project was prototyping and programming using p5.js and C#. Yiqi Zhao's contribution was rendering.
